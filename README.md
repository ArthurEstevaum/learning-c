# learning-c
Repositório de exercícios e exemplos em linguagem C para aprendizado

# Executando código
Para executar qualquer um dos programas neste repositório é necessário um compilador de linguagem C.
Durante meus exercícios usei o GCC.
Observação: Programas que incluem bibliotecas como math.h necessitam ser incluidas com -l ao executar a compilação.
